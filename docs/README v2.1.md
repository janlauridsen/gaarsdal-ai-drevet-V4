# AI-drevet faglig afklaring for hypnoterapi

## Status (bindende)
**Designfase afsluttet – v2.1 fastlåst**  
Projektet befinder sig nu mellem **færdigt design** og **reference-implementering**.

Denne repository indeholder **normative dokumenter**, testgrundlag og driftsprincipper
for et produkt, der leverer **faglig afklaring uden rådgivning, behandling eller relation**.

Implementering må **udelukkende reproducere** den fastlagte adfærd.

---

## Formål

Projektets formål er at skabe en AI-drevet faglig afklaringsoplevelse for brugere,
der overvejer hypnoterapi.

Produktet skal:
- reducere uklarhed
- øge faglig præcision
- tydeliggøre grænser

Produktet skal **ikke**:
- rådgive
- behandle
- guide brugeren mod handling
- fungere relationelt eller terapeutisk

---

## Produktets karakter

Produkt-AI fungerer som:
- et fagligt afklaringslag
- en strukturel grænsemarkør
- en begrebslig præciseringsmekanisme

Produkt-AI er **bevidst ikke**:
- hjælpsom i handlingsforstand
- trøstende
- håbsforstærkende
- validerende

Oplevet merværdi opstår gennem:
- forklaring af *hvorfor* spørgsmål ikke kan besvares
- tydelige faglige skel
- sproglig lukning uden rådgivning
